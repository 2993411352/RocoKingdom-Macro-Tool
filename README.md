# 洛克王国：世界 — 进阶视觉宏自动化辅助 (Python)

基于原生 Python 打造的《洛克王国：世界》全自动挂机方案。本项目彻底抛弃了传统“死心眼式”的固定点击脚本，融入了**实时视觉感知**模块。在自动化执行复杂打怪与采集连招的同时，赋予了程序**自动清理干扰弹窗、智能解除被冰冻/果冻状态、后台窗口失焦自保**等多项高级生存能力。

## ✨ 核心特性

- 👁️ **双轨视觉神经** 
  - **防弹窗机制**：自动识别并清理意外打开的界面或系统层级弹窗，且**绝不打断当前连招进度**。
  - **解除控制机制**：检测到“果冻/冰封”状态后，瞬间变身电竞手速狂按解除键，破冰后自恢复至原挂机路线。
- 🪟 **窗口焦点检测** — 随时切出干别的事，程序一旦发现游戏失去焦点会光速暂停，切回来自动接着打，不怕鼠标失控乱点。
- 🔄 **智能重置防呆** — 自带 5 分钟安全重置机制，定时触发补发精灵（应对召唤物消失机制）。
- 📝 **极简配置中心** — `config.py` 和 `presets.py` 界面分离，小白也能在三分钟内自定义自己的神仙连招。

## 🚀 极速上手 (纯净版玩家看这里)

如果你不想在电脑上部署一行代码环境，只想单纯拿来用，这是为你准备的：

1. 进入本项目 GitHub 右侧的 **[Releases](https://github.com/2993411352/RocoKingdom-Macro-Tool/releases)** 页面。
2. 下载最新的 `RocoKingdom-Macro-Release.zip` 压缩包。
3. **[重要]** 解压后，你会得到一个 `RocoKingdom-Macro.exe` 可执行文件和一个非常关键的 `templates/` 素材库文件夹（千万不要删除它）。
4. **右键 -> 并且一定要以管理员身份运行** `RocoKingdom-Macro.exe`，根据终端菜单进行连招挑选即可！

### 🎨 我的分辨率识别不到怎么办？
由于各种电脑屏幕尺寸和游戏内 UI 缩放不同，如果在你的电脑上脚本成了瞎子，你可以：
- 去游戏里遭遇对应场景（果冻/多余弹窗），用自己的 QQ 截图截下那个小小的标识图。
- 把它起名为 `jelly_template.png` (果冻) 或 `close_template.png` (弹窗红叉)。
- 覆盖掉 `templates/` 文件夹下原来的图，再次启动 `exe` 就能完美适配你的电脑了。

## 🛠️ 开发者部署指南 

如果你懂 Python，想在源码基础上进行魔改和贡献：

### 环境部署
```bash
git clone https://github.com/2993411352/RocoKingdom-Macro-Tool.git
cd RocoKingdom-Macro-Tool
pip install -r requirements.txt
```

### 核心架构（你的手术台）
本项目已完成彻底的高内聚解耦，目录结构极其清爽：

```text
RocoKingdom-Macro-Tool/
├── core/                   # 🚫 核心引擎黑盒 (禁止小白乱动)
│   ├── macro_engine.py     # 状态机中枢与调度系统
│   ├── command_parser.py   # 自研延迟指令组建解析器
│   ├── vision_detector.py  # OpenCV 视觉并发监听
│   └── window_detector.py  # Windows API 前景窗口捕捉
├── debug/                  # 🔍 自修复标定工具箱
│   ├── debug_roi.py        # ROI区域画框可视化(可测试你的截取范围)
│   └── test_vision.py      # OpenCV 置信度实时测谎仪
├── templates/              # 🖼️ 目标素材图库
│   ├── close_template.png
│   └── jelly_template.png
├── config.py               # ⭐ 【主要战场】宏延迟、键位、模板范围总控配置
├── presets.py              # ⭐ 【连招大本营】编写复杂按键宏的地方
└── main.py                 # 🟢 Python 用户原生启动入口
```

### 将改动进行 EXE 二次打包
如果你对代码进行了调整或重构，使用下面的命令一键构建新的成品单体分发文件：
```bash
pyinstaller --onefile --noconsole --icon=NONE --name "RocoKingdom-Macro" main.py
```

## 🎮 内置指令字典

在 `presets.py` 中编写属于你的终极打法，请参考下表自由排列组合（以空格分开）：

| 语法字符 | 含义与功能 |
|------|------|
| `LMouseDown` / `LMouseUp` | 鼠标左键按下 / 释放 |
| `WheelUp` / `WheelDown` | 全自动滚轮上推 / 下滚 |
| `QDown`/`QUp` & `XDown`/`XUp` | 所有的英文字母原声触发与松开 |
| `ShiftDown` / `ShiftUp` | Shift 按键管理 |
| `SpaceDown` / `SpaceUp` | 空格按键管理 |
| `Esc` / `Tab` | 瞬移长按敲击全过程 |
| `C1` ~ `C6` | 战术精灵 1 至 6 号自动切位 |
| `Bow` | 高级原生交互：脱战鞠躬 |
| `Sound` | 系统提示音 `咚` |
| `500` / `1500` | 延迟休息 500ms 或 1500ms (推荐灵活穿插) |
