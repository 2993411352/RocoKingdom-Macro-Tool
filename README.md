# 洛克王国：世界 — 自动化宏工具

基于 Python 的《洛克王国：世界》全自动挂机方案。支持自定义动作场景与 **视觉防卡死机制**（自动解除果冻状态、自动关闭干扰弹窗），实现真正的无干预后台挂机。

## 🎯 核心特性

- 🎮 **全局热键控制** — `Ctrl+Space` / `Alt+Space` 一键启动/停止
- 🪟 **窗口焦点检测** — 游戏失焦自动暂停，切回后自动恢复动作
- 🔄 **智能定时重置** — 自带 5 分钟安全重置机制，补发全队精灵
- 👁️ **图像视觉纠错** — 受到控制（果冻）自动按 X 解除；遭遇弹窗遮挡自动定位关闭
- 📝 **极简配置中心** — `config.py` 和 `presets.py` 界面分离，可根据需求极速自定义新动作

## 🚀 下载与使用

### 1. 快速使用 (推荐)
1. 在 [Releases](https://github.com/2993411352/RocoKingdom-Macro-Tool/releases) 页面下载最新的 `RocoKingdom-Macro-Release.zip`。
2. 解压后，保持 `templates/` 文件夹与可执行程序在同级目录。
3. **以管理员身份运行** `RocoKingdom-Macro.exe`。
4. 终端会弹出可用预设菜单，输入编号启动对应的动作序列。

### 2. 开发者源码运行
```bash
git clone https://github.com/2993411352/RocoKingdom-Macro-Tool.git
pip install -r requirements.txt
python main.py
```

## 🎮 当前可用预设 (Presets)

启动后会弹出交互式菜单供选择：

| 编号 | 预设名称 | 执行逻辑 |
|------|------|------|
| 1 | 终端自定义 | 直接在控制台手动输入你的专属动作序列 |
| 2 | 主座设置 | 扔出2~6号精灵后骑乘1号，每20秒按R防掉线 |
| 3 | 纯鞠躬小号设置 | 扔出1~6号精灵后，持续循环: Tab→鞠躬→Esc |
| 4 | 单人采集设置 | 扔出2~6号精灵，循环: Tab→鞠躬→Esc→跳跃→ RDown → RUp |

*(如有需要，您可以使用记事本打开并编辑 `presets.py` 任意增减修改这些循环逻辑)*


## ⌨️ 内置指令参考

在编写自己的专属动作时，请使用空格分隔以下指令：

| 指令语法 | 触发效果 |
|------|------|
| `LClick` / `RClick` | 发送一次鼠标左/右键点击 |
| `HoldLClick:1000` | 长按鼠标左键 1000 毫秒后松开（可替换任意键如 `HoldR:200`） |
| `WheelUp` / `WheelDown` | 滚轮上推 / 滚轮下拉 |
| `X` / `R` / `Space` / `Esc` | 解析为敲击按键（自动包含按下与抬起动作） |
| `QDown`/ `XUp` | 分离控制特定按键的按下与抬起 |
| `C1` ~ `C6` | 切换 1-6 号位对应的精灵 |
| `Bow` | 预设动作：鞠躬 |
| `Sound` | 系统内播放一声 `咚` 的提示音 |
| `800` / `1500` | 休眠等待 800ms 或 1500ms |
