# 洛克王国：世界 — Python 自动化工具

基于 Python的《洛克王国：世界》自动化宏工具，支持键鼠模拟、预设场景执行与**视觉检测自动恢复**。

## 功能特性

- 🎮 **全局热键控制** — `Ctrl+Space` / `Alt+Space` 一键启动/停止
- 🪟 **窗口焦点检测** — 失焦自动暂停，恢复后自动继续
- 📋 **指令解析引擎** — 完整支持原项目全部宏指令
- 🔄 **四套预设模板** — 单人6只奇丽X / 多人骑乘 / 纯鞠躬 / 自定义
- 👁️ **视觉检测 (新增)** — 自动识别果冻状态，按 X 解除后继续任务
- 🛡️ **智能恢复** — 窗口失焦/果冻恢复后自动重置 UI 状态

## 安装

```bash
pip install -r requirements.txt
```

**依赖列表:**
- `pydirectinput` — DirectInput 级别键鼠模拟
- `keyboard` — 全局热键监听
- `mss` — 高速屏幕截图
- `opencv-python` — 图像模板匹配
- `numpy` — 数组处理

## 使用方法

### 1. 启动工具

```bash
# 建议以管理员身份运行
python main.py
```

### 2. 选择预设

启动后会显示交互式菜单，选择对应编号即可：

| 编号 | 预设 | 说明 |
|------|------|------|
| 1 | 自定义连招 | 手动输入宏指令 |
| 2 | 单人6只奇丽X | 扔出6只精灵 + 自动鞠躬循环 |
| 3 | 多人全程骑乘 | 扔出精灵后骑乘，每20秒按R防掉线 |
| 4 | 多人纯鞠躬 | 扔出6只精灵 + 持续鞠躬 |

### 3. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Space` | 启动 / 停止宏 |
| `Alt + Space` | 备用启动 / 停止 |
| `F12` | 退出程序 |

### 4. 设置果冻检测模板

首次使用果冻检测功能前，需要截取一张参考图片：

1. 在菜单中选择 `[T] 截取果冻状态模板图片`
2. 将游戏切换到果冻状态画面
3. 按回车键截取

截取的图片会保存到 `templates/jelly_template.png`。

## 指令参考

| 指令 | 功能 |
|------|------|
| `LMouseDown` / `LMouseUp` | 鼠标左键按下 / 释放 |
| `WheelUp` / `WheelDown` | 滚轮上 / 下 |
| `QDown`/`QUp` ... `XDown`/`XUp` | 字母键按下 / 释放 |
| `ShiftDown` / `ShiftUp` | Shift 键 |
| `SpaceDown` / `SpaceUp` | 空格键 |
| `Esc` / `Tab` | 按下并释放 |
| `C1` ~ `C6` | 精灵切换（数字键 1-6） |
| `Bow` | 鞠躬 |
| `Press1280-720-0` | 绝对坐标点击 |
| `Sound` | 提示音 |
| `500` | 延迟 500 毫秒 |

## 文件结构

```
├── main.py              # 程序入口
├── config.py            # 全局配置
├── window_detector.py   # 窗口检测
├── command_parser.py    # 指令解析引擎
├── macro_engine.py      # 宏执行引擎
├── vision_detector.py   # 视觉检测
├── presets.py           # 预设模板
├── requirements.txt     # 依赖
└── templates/           # 模板图片
    └── jelly_template.png
```

## 注意事项

1. **管理员权限** — 必须以管理员身份运行，否则游戏窗口无法接收虚拟按键
2. **分辨率适配** — 坐标映射基准为 2560×1440，不同分辨率会自动缩放
3. **操作延迟** — 所有指令间保留合理延迟，防止操作过快被检测
4. **模板精度** — 果冻检测模板截图越清晰精确，识别越准确
